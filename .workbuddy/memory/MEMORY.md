# MEMORY.md - 项目长期记忆

## 项目概览
- 中科大 AED 急救引导网页，单页应用（无框架）
- 文件结构：`index.html` + `app.js` + `data.js`，照片位于 `照片/校区名/楼宇名/1.jpg ...`
- `data.js` 由 `generate_data.py` 从 Excel（`中科大AED引导分工与命名规范表.xlsx`）自动生成

## 关键约定
- `imageBasePath` 应为 `"照片"`（相对于 index.html 所在目录），不是 `"../照片"`
- 每次更新照片后，运行 `generate_data.py` 即可自动刷新 `data.js` 中的图片列表
- 无AED楼宇需同时展示：前往指引文字 + 目标楼宇的AED位置描述 + 目标楼宇的引导照片（app.js 已实现，依赖 data.js 中 nearestAED 楼宇有 images 数据）
